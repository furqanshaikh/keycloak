# Module 1: Keycloak Fundamentals and Architecture


**Duration:** 8–10 hours
**Level:** Advanced (Enterprise Foundations)


## Learning Objectives


By the end of this module, you will be able to:


- Understand enterprise IAM concepts
- Explain OAuth 2.0, OIDC, and SAML fundamentals
- Understand Keycloak architecture
- Deploy a development-grade Keycloak instance
- Choose correct OAuth/OIDC flows for applications


## Enterprise Scenario – TechCorp


TechCorp is a B2B SaaS company building a multi-tenant platform. They require:


- Centralized authentication
- SSO across applications
- Secure APIs and microservices
- A foundation for MFA, LDAP, and SAML


This module establishes the foundation for all advanced Keycloak topics.
