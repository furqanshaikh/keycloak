# Keycloak Installation Options


## Deployment Models


- Standalone (development)
- Docker / Docker Compose
- Kubernetes (Helm / Operator)


## Database Options


- PostgreSQL (recommended)
- MySQL / MariaDB
- Oracle (enterprise)


## Dev vs Production


| Development | Production |
|-----------|------------|
| Embedded DB | External DB |
| HTTP | HTTPS |
| Single node | Clustered |
